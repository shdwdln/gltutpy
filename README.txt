
This repo contains a Python translation of the C samples from the Modern
OpenGL tutorial at
http://www.arcsynthesis.org/gltut (online docs)
and
https://bitbucket.org/alfonse/gltut/overview (repo)


Requirements
------------

Python 2.7
pyglet 1.1.4
PyOpenGL 3.0.1
